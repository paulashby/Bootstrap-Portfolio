Open Triviata Database website

While working with data from the Open Trivia Database for a quiz app, I became frustrated by the lack of question ID numbers. As the inclusion of IDs would greatly simplify the handling of question data, I wrote a PHP programme to add the questions to my own database and an API to serve them along with IDs. The API homepage makes use of the UIKit css framework.

Role: Sole developer, designer
